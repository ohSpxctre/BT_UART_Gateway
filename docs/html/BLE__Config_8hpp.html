Detailed Description

Configuration file containing default values and structures for BLE server and client.

This header file defines constants, default parameters, and structures used for configuring the BLE (Bluetooth Low Energy) server and client. It includes settings for advertising, GATT (Generic Attribute Profile) server and client, UUIDs, MTU size, and other BLE-related parameters.

The file is organized into the following sections:

  • BLE Server Definitions: Contains default values and structures for the BLE server, including GATT service and characteristic configurations, advertising parameters, and server profile instance structure.
  • BLE Client Definitions: Contains default values and structures for the BLE client, including scan parameters, remote service and characteristic UUIDs, and client profile instance structure.
Note
This file is designed for use with the ESP-IDF framework and relies on its BLE APIs.
