◆ receiveTask()

void Uart::receiveTask ( MessageHandler *  msgHandler)

Task function to read characters from UART and buffer input.

Reads user input from UART, handling special characters like backspace, and sends complete messages to the data parser.

Reads character-by-character until a newline (\n) or carriage return (\r) is received. Handles backspace correctly by editing the input buffer and updating terminal output. Forwards the final message to the DATA_PARSER_QUEUE for parsing.

Parameters
msgHandlerPointer to MessageHandler used to enqueue parsed messages.

This function echoes characters as they arrive, supports editing via backspace, and detects end-of-line sequences (\r or \n) to terminate the input.
