Detailed Description

Implementation of the Uart class for UART communication on ESP32-C6.

This file defines the logic for the UART driver wrapper used in the application, including initialization, sending, and receiving data via FreeRTOS queues. It supports backspace handling, newline expansion, and buffering of UART data for further parsing or forwarding.

Note
This implementation is designed for use with the ESP-IDF framework.
The class supports optional expansion of \r/\n to \r\n sequences to ensure terminal compatibility.
The receive task echoes characters and supports backspace processing for interactive input.
